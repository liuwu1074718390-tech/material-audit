# 使用说明

## 功能概述

材价审计系统用于上传材料价格清单，通过AI分析给出价格建议和偏差报告。

## 快速开始

### 1. 上传Excel文件

点击主页的"上传文件"按钮，在弹出的窗口中：

- 拖拽Excel文件到上传区域，或点击上传区域选择文件
- 支持的文件格式：`.xls` 和 `.xlsx`
- 文件会自动解析，无需手动提交

### 2. Excel文件格式要求

Excel文件应包含以下列（从A列开始）：

| 列 | 字段名 | 类型 | 说明 |
|----|--------|------|------|
| A | 序号 | 数字/文本 | 材料序号 |
| B | 编码 | 文本 | 材料编码（必填） |
| C | 类别 | 文本 | 材料类别 |
| D | 名称 | 文本 | 材料名称 |
| E | 规格型号 | 文本 | 规格型号 |
| F | 单位 | 文本 | 计量单位 |
| G | 数量 | 数字 | 数量 |
| H | 不含税市场价 | 数字 | 单价（不含税） |
| I | 税率 | 数字 | 税率（%） |

**注意事项**：
- 第1行为表头，数据从第2行开始
- 编码字段为必填项，用于识别材料
- 数字字段应为纯数字，不要包含货币符号或其他文字

**示例数据**：

| 序号 | 编码 | 类别 | 名称 | 规格型号 | 单位 | 数量 | 不含税市场价 | 税率 |
|------|------|------|------|----------|------|------|--------------|------|
| 1 | 010101001001 | 钢筋 | 热轧带肋钢筋 | HRB400 Φ12 | t | 10.5 | 4200 | 13 |
| 2 | 010201002001 | 水泥 | 硅酸盐水泥 | P.O 42.5 | t | 50 | 380 | 13 |

### 3. 配置审计参数

文件上传成功后，点击"下一步"进入参数配置页面：

#### 地区（选填）
- 选择省份或城市
- 用于获取当地市场价格信息
- 示例：广东省 > 广州市

#### 时间范围（选填）
- 选择月份范围
- 用于分析特定时期的价格趋势
- 示例：2024-01 至 2024-12

#### 材料类别（选填）
- 可多选
- 选项从上传的Excel文件C列自动提取
- 只对选中的类别进行审计

### 4. 发起审计

配置完成后，点击"发起材价审计"按钮：

- 系统会调用AI工作流进行分析
- 分析过程可能需要几秒到几十秒
- 完成后自动显示审计结果

## 审计结果说明

### 结果列表

审计完成后，会显示包含以下字段的结果表格：

| 字段 | 说明 |
|------|------|
| 序号 | 材料序号 |
| 编码 | 材料编码 |
| 类别 | 材料类别 |
| 名称 | 材料名称 |
| 规格型号 | 规格型号 |
| 单位 | 计量单位 |
| 数量 | 数量 |
| 不含税市场价 | 上传的市场价 |
| 税率 | 税率 |
| 不含税市场价合计 | 价格合计 |
| 推荐价格范围(不含税) | AI建议的合理价格区间 |
| 价格偏差 | 与建议价格的偏差百分比 |

### 价格偏差说明

价格偏差用不同颜色标识：

- 🔴 **红色（偏高）**: 偏差 > +20%，价格明显高于市场
- 🟢 **绿色（正常）**: 偏差在 -20% ~ +20%，价格合理
- 🟡 **黄色（偏低）**: 偏差 < -20%，价格明显低于市场

### 筛选功能

使用筛选器可以快速定位关注的数据：

1. **按编码筛选**: 输入编码关键词
2. **按类别筛选**: 从下拉列表选择类别
3. **按名称筛选**: 输入名称关键词
4. **按规格型号筛选**: 输入规格型号关键词
5. **按价格偏差筛选**: 选择偏差范围
   - 偏高 (>20%)
   - 正常 (-20%~20%)
   - 偏低 (<-20%)

点击"查询"按钮应用筛选，点击"重置"清除筛选条件。

### 导出功能

点击"导出Excel"按钮可以将当前结果（包括筛选后的数据）导出为Excel文件。

导出的文件包含所有列，可以在Excel中进一步分析。

## 历史记录

系统会自动保存审计记录，在主页可以查看历史审计结果。

## 常见问题

### Q: 为什么文件上传失败？

A: 请检查：
- 文件格式是否为 .xls 或 .xlsx
- 文件是否损坏
- 文件大小是否超过限制（建议<10MB）

### Q: 为什么某些材料没有推荐价格？

A: 可能原因：
- 材料信息不完整（缺少名称、规格等）
- AI未能识别该材料类型
- 市场上该材料价格信息较少

### Q: 如何理解价格偏差？

A: 价格偏差 = (上传价格 - 推荐价格平均值) / 推荐价格平均值 × 100%

- 正值表示上传价格高于推荐价格
- 负值表示上传价格低于推荐价格
- 建议关注偏差绝对值>20%的材料

### Q: 推荐价格范围的依据是什么？

A: 推荐价格基于以下因素：
- 所选地区的市场价格
- 所选时间段的价格趋势
- 同类材料的历史价格
- AI模型的综合分析

### Q: 可以同时上传多个文件吗？

A: 当前版本仅支持单文件上传。如需审计多个清单，请分别上传。

### Q: 审计结果可以保存吗？

A: 可以。每次审计的结果都会自动保存，可以在历史记录中查看，也可以导出为Excel文件本地保存。
