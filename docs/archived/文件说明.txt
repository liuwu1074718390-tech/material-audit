1. 阅读「交付说明.md」了解部署流程
2. 运行 ./verify.sh 验证环境
3. 执行 npm install 安装依赖
4. 复制 .env.example 为 .env 并配置
5. 运行 npm run build 构建项目
6. 使用 PM2 或 Docker 启动服务
