# Node.js 安装步骤（详细图文版）

## 🎯 推荐方法：官网下载安装（最简单）

### 第一步：打开 Node.js 官网

1. 打开您的浏览器（Safari、Chrome 等）
2. 在地址栏输入：`https://nodejs.org/`
3. 按回车键访问

### 第二步：下载安装包

在 Node.js 官网首页，您会看到两个大按钮：

```
┌─────────────────┐  ┌─────────────────┐
│   18.19.0 LTS   │  │  21.5.0 Current │
│  (推荐大多数用户) │  │   (最新特性)    │
└─────────────────┘  └─────────────────┘
```

**请点击左边的 "LTS" 按钮**（绿色的那个）

- LTS = Long Term Support（长期支持版）
- 更稳定，推荐使用
- 会自动下载适合您 Mac 的安装包

### 第三步：找到下载的文件

1. 打开 Finder（访达）
2. 点击左侧的"下载"文件夹
3. 找到刚下载的文件，名称类似：
   ```
   node-v18.19.0.pkg
   ```

### 第四步：安装 Node.js

1. **双击** `.pkg` 文件
2. 会弹出安装向导窗口
3. 按照提示操作：

```
步骤 1: 欢迎界面
   → 点击"继续"

步骤 2: 许可协议
   → 点击"继续"
   → 点击"同意"

步骤 3: 安装位置
   → 保持默认，点击"继续"

步骤 4: 安装类型
   → 点击"安装"
   → 输入您的 Mac 密码
   → 点击"安装软件"

步骤 5: 安装中...
   → 等待进度条完成（约1-2分钟）

步骤 6: 完成
   → 点击"关闭"
```

### 第五步：验证安装

1. **关闭当前的终端窗口**（重要！）
2. **重新打开终端**
3. 在终端中输入以下命令：

```bash
node -v
```

应该显示类似：
```
v18.19.0
```

再输入：
```bash
npm -v
```

应该显示类似：
```
9.2.0
```

**如果看到版本号，恭喜您，安装成功！** 🎉

### 第六步：启动项目

现在可以运行项目了：

```bash
# 1. 进入项目目录
cd /Users/liuwu/Desktop/材价审计

# 2. 安装项目依赖
npm install

# 3. 启动开发服务器
npm run dev
```

等待启动完成后，在浏览器中访问：
```
http://localhost:3000
```

## ❓ 常见问题

### Q1: 安装后仍然提示 "command not found"

**原因**：终端还在使用旧的环境变量

**解决方案**：
1. 完全关闭终端应用
2. 重新打开终端
3. 再次运行 `node -v`

### Q2: 下载速度很慢

**解决方案**：
- 耐心等待，安装包约 30-50MB
- 或者使用国内镜像站下载：
  - 访问：`https://npmmirror.com/mirrors/node/`
  - 找到对应版本的 `.pkg` 文件下载

### Q3: 安装时提示"无法打开，因为它来自身份不明的开发者"

**解决方案**：
1. 按住 Control 键，点击 `.pkg` 文件
2. 选择"打开"
3. 在弹出的对话框中点击"打开"

### Q4: 需要输入密码但不知道是什么密码

**说明**：
- 这是您的 **Mac 登录密码**
- 就是开机时输入的密码
- 输入时不会显示任何字符（这是正常的）
- 输入完成后按回车键

### Q5: 想要卸载 Node.js

**解决方案**：
如果需要重新安装或卸载，运行：

```bash
# 删除 Node.js
sudo rm -rf /usr/local/bin/node
sudo rm -rf /usr/local/bin/npm
sudo rm -rf /usr/local/lib/node_modules
sudo rm -rf /usr/local/include/node
sudo rm -rf /usr/local/share/man/man1/node.1
```

## 🎯 安装成功的标志

当您在终端运行 `node -v` 和 `npm -v` 都能看到版本号时，就表示安装成功了！

示例输出：
```bash
$ node -v
v18.19.0

$ npm -v
9.2.0
```
